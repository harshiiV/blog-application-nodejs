# blog-application-nodejs
# (Node.js + Express + EJS)

A full-stack **blog web application** built using **Node.js**, **Express.js**, and **EJS** templates.  
It features user authentication, comment functionality, and a responsive front-end design.

---

##  Features
- Admin login system (secure access to blog pages)
- Dynamic blog posts and content rendering
- Comment section with persistent storage (JSON-based)
- Contact form that saves messages
- Responsive, clean UI with a light blue theme ‚ú®

---

##  Tech Stack
- **Backend:** Node.js, Express.js  
- **Frontend:** HTML, CSS, EJS Templates  
- **Storage:** JSON (File-based)  
- **Tools:** Git, VS Code
